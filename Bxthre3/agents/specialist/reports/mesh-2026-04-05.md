## 4. State Sync & Conflict Resolution

### Resource Leasing
```
TTL-based leases prevent concurrent writes:
- read:  multiple readers OK
- write: exclusive once acquired
- exclusive: blocks all others
```

### EventBus (Python)
- 500-event circular buffer
- Event types: task.created, task.completed, rating.updated, agent.degraded, alert.created
- Pub/sub with subscriber management
- **Gap:** No disk persistence — events lost on restart

### MCPMesh EventBus (TypeScript)
- Pattern-based subscriptions
- Broadcast to authed peers only
- **Gap:** In-memory only, no persistence
