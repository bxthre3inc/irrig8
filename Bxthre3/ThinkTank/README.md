# ThinkTank

Draft product specifications that have not yet been promoted to official specs.

## File Naming Convention
- 6-7 character codes only
- Valid: IRRIG8, AUTODRK, VALPLAY1, BIGBOX01
- Invalid: IRR08 (too short), IRRIGATION (too long)

## Status
- `draft`: Initial creation, fields may be [TBD]
- `ready`: All required fields completed, awaiting promotion
- `archived`: Deprecated drafts (date suffix added)

## Required Fields for Promotion
All [TBD] fields must be resolved before moving to /specs/
