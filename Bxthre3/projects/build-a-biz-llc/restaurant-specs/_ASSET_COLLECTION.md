# Asset Collection Workflow
## Build-A-Biz Restaurant Specs

**Purpose**: Systematically collect branding assets for all restaurant leads

---

## Collection Checklist Per Restaurant

### Phase 1: Brand Identity
- [ ] **Logo** (PNG, SVG preferred)
- [ ] **Primary Color** (hex code)
- [ ] **Secondary Color** (hex code)
- [ ] **Accent Color** (hex code)
- [ ] **Font Preferences** (heading + body)

### Phase 2: Photography
- [ ] **Hero Image** (storefront, exterior)
- [ ] **Interior Ambiance** (2-3 shots)
- [ ] **Signature Dishes** (6-12 items)
- [ ] **Owner/Team Photo** (for "About" page)

### Phase 3: Menu Data
- [ ] **Full Menu** (PDF or website)
- [ ] **Item Names** (exact spelling)
- [ ] **Item Prices**
- [ ] **Item Descriptions**
- [ ] **Dietary Tags** (V, VG, GF, etc.)
- [ ] **Allergen Info**

### Phase 4: Business Info
- [ ] **Full Address**
- [ ] **Hours of Operation**
- [ ] **Phone Number(s)**
- [ ] **Email**
- [ ] **Google Maps URL**
- [ ] **Social Media Links**

---

## Collection Methods

### Method 1: Owner Outreach (Best)
Contact restaurant directly and request:
- Press kit / media folder
- High-res logo file
- Menu PDF
- Professional photos (if available)

### Method 2: Web Scraping (Fast)
- Restaurant website (logo, colors, menu)
- Google Business Profile (photos, hours)
- Yelp (menu items, prices, reviews)
- Facebook/Instagram (brand aesthetic)

### Method 3: In-Person (Most Accurate)
- Visit restaurant
- Photograph menu boards
- Photograph food items
- Capture brand colors from signage
