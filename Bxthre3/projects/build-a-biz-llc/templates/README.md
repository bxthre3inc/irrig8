# Build-A-Biz App Templates
## Config-Driven Restaurant Ordering App Generator

Generate complete custom ordering apps (iOS, Android, Web) from a single `.config` file + assets folder.

---

## Quick Start

```bash
# 1. Pick a template base
cd templates/modern-minimal

# 2. Edit your config
cp EXAMPLE.config.json my-restaurant.config.json
# Fill in: name, menu, UberEats/DoorDash keys, Stripe keys

# 3. Add assets to assets/
# - logo.png (512x512)
# - hero.jpg (1920x1080)
# - menu/ (category photos)

# 4. Build the app
bun generate.ts my-restaurant.config.json

# 5. Output appears in dist/my-restaurant/
# Capacitor sync → Xcode & Android Studio
```

## Config Architecture

```json
{
  "meta": { /* branding */ },
  "menu": { /* categories, items, modifiers */ },
  "integrations": { /* UberEats, DoorDash, Stripe, CRED */ },
  "loyalty": { /* rewards program config */ },
  "giftCards": { /* digital gift card settings */ },
  "features": { /* enable/disable features */ }
}
```

## Integration Checklist

Before running `generate.ts`, ensure you have:

### Uber Eats
- [ ] Signed partnership agreement
- [ ] Store UUID from Uber dashboard
- [ ] API key from Uber Eats Developer Portal

### DoorDash
- [ ] DoorDash Drive account
- [ ] Developer API credentials
- [ ] Store ID

### Stripe
- [ ] Stripe Connect account
- [ ] Publishable key
- [ ] Secret key (stored in env, not config)

### CRED Wallet (Optional)
- [ ] CRED merchant account
- [ ] Wallet API credentials
- [ ] Accepted currencies configured
