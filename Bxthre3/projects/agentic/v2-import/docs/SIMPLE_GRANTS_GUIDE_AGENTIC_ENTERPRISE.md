# Project Agentic Enterprise: Simple Grants Guide
## Least Glamorous, Highest Accessibility Funding Opportunities

---

## Executive Summary

This guide identifies grant opportunities that value **simple solutions to complex problems** with **minimal bureaucracy**. These are ideal for independent developers and small teams working on open-source AI projects.

**Important Note on Geography:** Many of the best simple grants happen to be European/UK-based. This is because Europe has invested heavily in "digital sovereignty" and "open internet" initiatives. However, most accept **global applicants** - you don't need to be European to apply.

---

## Tier 1: Highest Accessibility (Recommended Start)

### 1. FLOSS/fund ⭐⭐⭐⭐⭐ (GLOBAL - India-based)

**Why it fits Agentic Enterprise:**
- **$10,000 - $100,000 grants** for open source projects
- **Global** - accepts applications worldwide
- **Simplest application ever** - just add a `funding.json` file to your repo
- Funded by Zerodha (India's largest stock broker)
- $1 million committed annually
- No institutional affiliation required

**Application URL:** https://dir.floss.fund/submit

**What makes it unique:**
- Decentralized discovery - your `funding.json` file makes you discoverable
- No lengthy forms - structured manifest file approach
- Transparent - all applications published on dir.floss.fund

**Why it's "least glamorous":**
- No demo days, no accelerator, no equity
- Just: "Here's money for your open source work"
- The application is literally a JSON file in your repo

### 2. NLnet Foundation / NGI Zero Commons Fund ⭐⭐⭐⭐⭐ (EU-based, Global applicants)

**Why it fits Agentic Enterprise:**
- **Rolling deadlines** (next: June 1, 2026)
- **Simple one-page application** - just fill out a web form
- **No institutional affiliation required**
- **Open to individuals worldwide**
- Typical grants: **€20,000 - €50,000**
- Focus: Open source, privacy-preserving, decentralized technologies

**Application URL:** https://nlnet.nl/propose/

**What they've funded:**
- AI Horde (collaborative AI inference)
- Many privacy-focused open source projects

**Why it's "least glamorous":**
- No flashy announcements or demo days
- Just straightforward funding for useful open source work
- They explicitly say "Don't waste too much time on attachments. Really."

**Deadline:** April 6, 2026

**Requirements:**
- Must be maintainer of at least 3 FOSS projects
- Projects must meet "criticality" criteria

- Not about flashy new products, but sustainable infrastructure
- Work that's often invisible but essential

## Application Strategy for Agentic Enterprise

### Primary Target: NLnet + AI Grant

**Why these two first:**
1. **Simplest applications** - web forms, minimal attachments
2. **No institutional requirements** - individuals can apply
3. **Fastest turnaround** - rolling deadlines
4. **Perfect alignment** - on-device AI, privacy, open source

### NLnet Application Framing:

**Project Title:** Agentic Enterprise: On-Device Continual Learning for Resource-Constrained Devices

**Abstract (draft):**
Agentic Enterprise demonstrates the first practical on-device continual learning system for sub-$200 Android smartphones. While Apple and Google focus on inference, Agentic Enterprise enables AI to learn and adapt from user interactions entirely offline, using LoRA-style adapter updates and a minimal skill evolution architecture. This addresses the "last mile" of AI accessibility - bringing intelligent, privacy-preserving agents to the 6.8 billion smartphone users worldwide, including those with budget devices and limited connectivity.

**Technical Challenges:**
1. On-device learning with <500MB RAM
2. LoRA adapter updates without cloud connectivity
3. Skill evolution through self-play on mobile hardware
4. SMS-based interface for universal accessibility

**Ecosystem:**
- Open source all code under MIT license
- Publish research findings
- Build community of contributors
- Create benchmark for on-device learning

## Key Insight: "Least Glamorous" = Highest Success Rate

The grants in Tier 1 share common traits:

1. **Simple applications** - hours to prepare, not weeks
2. **No credentials required** - work speaks for itself
3. **Rolling deadlines** - no waiting for annual cycles
4. **Individual-friendly** - no university or company needed
5. **Clear criteria** - you'll know if you fit before applying

These funders understand that **bureaucracy is a barrier to innovation**. They've designed their processes to get money to developers quickly, with minimal overhead.
