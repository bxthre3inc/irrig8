# SLV Mesh — San Luis Valley Free Communications Network

> Free, off-grid text messaging for the San Luis Valley using LoRa mesh radio.

**Status:** 🟡 Pilot — Low Activity  
**Owner:** Unassigned  
**Last Updated:** 2026-03-25

## What
A community-owned mesh network covering the San Luis Valley. When cell towers fail, neighbors still talk.

## Why
Rural communities are vulnerable when infrastructure goes down. Wildfires, storms, and isolation cut off communication. SLV Mesh provides a free, resilient alternative.

## How
- **Protocol**: Meshtastic (LoRa mesh, ESP32 + 915MHz radio)
- **No internet required**: Pure radio mesh
- **No ongoing cost**: Hardware only ($85-120/node)
- **Encrypted**: End-to-end private
- **Open**: Anyone can join the network

## Architecture
P2P mesh across 11 planned nodes spanning Antonito → South Fork through the valley.
