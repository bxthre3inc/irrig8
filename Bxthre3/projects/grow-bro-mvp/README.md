# Grow Bro MVP

A plant growth tracking and care assistant.

## Features

- **Plant Profiles**: Create and manage plant profiles with species, growth stage, and care needs
- **Care Schedule**: Track watering, fertilizing, and pruning schedules
- **Growth Logging**: Photo and note logging for growth tracking
- **Reminders**: Automated care reminders based on plant needs
- **Dashboard**: At-a-glance view of all plants and upcoming care tasks

## Architecture

- Frontend: React + TypeScript + Tailwind CSS (Zo Space)
- Backend: API routes for data persistence
- Storage: In-memory with localStorage fallback (MVP)

## Quick Start

Deployed on Zo Space at `/grow-bro` routes.
