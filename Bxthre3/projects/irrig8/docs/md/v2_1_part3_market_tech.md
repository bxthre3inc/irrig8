## 2.2 Competitive Moat: Determinism vs. Stochastic Estimation

### 2.2.1 Competitor Analysis Matrix

| Competitor | Approach | Strengths | Weaknesses | FarmSense Advantage |
|------------|----------|-----------|------------|---------------------|
| **CropX** | Cloud-only soil analytics | Low upfront cost, easy install | Requires cellular, no edge autonomy, cloud latency | Edge autonomy, works offline, sovereign ledger |
| **FieldNET** | Pivot control systems | Proven hardware, wide install base | Pivot-only, no soil sensing, no VRI | Full SPAC modeling, soil + plant + atmosphere |
| **Arable** | Weather stations + vision | Good weather data, simple | Ground sensors limited, no irrigation control | Ground-truth sensor fusion, direct actuator control |
| **Prospera (Valmont)** | Computer vision only | No field hardware | Weather-dependent, no subsurface data | 48" deep-profile truth, 3D soil moisture grid |
| **Taranis** | Drone scouting | High-res imagery, pest detection | Episodic (not continuous), expensive per-acre | Continuous monitoring, 15-minute updates |
| **Satellite-only** (Ceres, Hummingbird) | Remote sensing | No field installation, broad coverage | 30m resolution, cloud gaps, no ground truth | 1m resolution, ground-truth correction, all-weather |
