# Partner Performance Reviews

**Cadence:** Quarterly

## Review Framework

### Scorecard Metrics

| Metric | Weight | Description |
|--------|--------|-------------|
| Revenue Contribution | 30% | Total ACV attributed to partner |
| New Logo Acquisition | 25% | Net new customers sourced |
| Deal Registration Compliance | 20% | Deals properly registered |
| Training Certification | 15% | Staff completed product certs |
| Co-Marketing Participation | 10% | Joint campaigns executed |

### Scoring Scale

| Score | Rating | Definition |
|-------|--------|------------|
| 90–100 | Elite | Exceeding all benchmarks |
| 75–89 | Strong | Consistently meeting + exceeding |
| 60–74 | Satisfactory | Meeting minimum requirements |
| 45–59 | At Risk | Below threshold — improvement plan |
| < 45 | Probation | Termination risk — corrective action |

### Review Agenda Template

1. **Revenue recap** — YTD performance vs. target
2. **Pipeline review** — Open opps + forecast
3. **Win/loss analysis** — Top wins, key losses
4. **Enablement gap** — Training, collateral, support
5. **Strategic alignment** — 90-day objectives
6. **Comp/rebate status** — Eligibility confirmation
7. **Action items** — Owner + due date
