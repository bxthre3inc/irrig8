## Stage Definitions
1. **Prospect** — Identified, not contacted
2. **Contacted** — Outreach sent, no response yet
3. **Engaged** — Responded, in conversation
4. **Qualified** — Fit confirmed, moving to terms
5. **Contracted** — Agreement signed
6. **Active** — Live and transacting
7. **Churned** — Terminated / inactive
