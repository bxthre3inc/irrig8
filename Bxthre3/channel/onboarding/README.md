# Partner Onboarding & Enablement

## Standard Onboarding Checklist

### Phase 1 — Pre-Sign (Days 0–3)
- [ ] Prospect identified and qualified
- [ ] Discovery call completed
- [ ] Partner type + tier determined
- [ ] NDA executed (if required)
- [ ] Preliminary terms agreed

### Phase 2 — Contract (Days 4–7)
- [ ] Partner Agreement executed
- [ ] Partner tier / commission rate set
- [ ] Reseller certs / training access granted
- [ ] CRM contact created
- [ ] Stripe partner account configured

### Phase 3 — Enable (Days 8–14)
- [ ] Product training scheduled / completed
- [ ] Marketing collateral delivered
- [ ] Demo environment access provided
- [ ] Co-sell motion established
- [ ] First live transaction assisted

### Phase 4 — Scale (Day 15+)
- [ ] Monthly business review scheduled
- [ ] Performance tracked against SLA
- [ ] Rebate / bonus eligibility confirmed

## Partner Tiers

| Tier | Criteria | Commission | Rebate |
|------|----------|------------|--------|
| Bronze | < $50K ACV | 15% | — |
| Silver | $50K–$250K ACV | 20% | 5% quarterly |
| Gold | $250K–$1M ACV | 25% | 10% quarterly |
| Platinum | > $1M ACV | 30% | 15% quarterly + MDF |
