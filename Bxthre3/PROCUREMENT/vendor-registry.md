## Onboarding Workflow

```
1. Sourcing Request → RFQ / RFP issued
2. Vendor Selection → Quote comparison → Approval
3. Contract Negotiation → Legal review → Sign-off
4. Onboarding → Set up in registry → Issue PO
5. Ongoing Management → Quarterly review → Renewal/Offboard
```

---

## Offboarding Workflow

```
1. Termination Notice → Confirm contract end date
2. Data & Access Revocation → Credentials, SSO, API keys
3. Final Invoice Reconciliation → Close PO
4. Offboard in Registry → Move to Offboarded section
5. Documentation → Archive contracts, notes
```
