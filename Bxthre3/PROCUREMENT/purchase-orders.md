## PO Workflow

```
1. Request → Department submits need (what, why, how much)
2. Approval → Purchase verifies budget + necessity
3. Issue PO → Formal order to vendor
4. Fulfillment → Vendor delivers goods/services
5. Receipt → Department confirms receipt
6. Reconciliation → Ledger matches invoice to PO
7. Close → PO marked complete, archived
```
