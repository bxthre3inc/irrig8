# Agent Onboarding Checklist
**For all new AgentOS agents**

---

## Day 1 — Foundation

### Before You Start
- [ ] Added to org chart (`the-agentos-project/core/hierarchy/org.ts`)
- [ ] Created INBOX file at `Bxthre3/INBOX/agents/{agent}.md`
- [ ] Added to appropriate department INBOX
- [ ] Assigned department lead
- [ ] Access to zo.space confirmed

### Workspace Orientation (1 hour)
- [ ] Read `SOUL.md` — behavioral identity, tone, rules
- [ ] Read `AGENTS.md` — routing, roster, projects
- [ ] Review `INBOX/agents/INBOX_ROUTING.py` — routing protocol
- [ ] Locate your department INBOX (`INBOX/departments/{dept}.md`)

### Architecture Briefing (2 hours)
- [ ] Read `the-agentos-project/AGENTOS_ARCHITECTURE_v6.2.md`
- [ ] Understand Determinism Principle
- [ ] Know your role in the Task Loop
- [ ] Understand trigger taxonomy

### Tools Setup
- [ ] Zo Computer access confirmed
- [ ] Gmail connected (if needed)
- [ ] Google Calendar connected (if needed)
- [ ] Linear connected (if applicable)
- [ ] Notion connected (if applicable)
- [ ] Airtable connected (if applicable)

---

## Day 2–3 — Role Immersion

### Department Training
- [ ] Complete department curriculum module E1/O1/SM1/GL1 (as applicable)
- [ ] Meet with department lead for role-specific context
- [ ] Review existing department INBOX backlog

### Your First Task
- [ ] Pick up a P2 or P3 from Honey-Do backlog
- [ ] Complete task and route result through INBOX
- [ ] Get feedback from department lead

---

## Day 4–5 — Integration

### Cross-Department Awareness
- [ ] Shadow another department's standup
- [ ] Understand how your work feeds other departments
- [ ] Know who your internal customers are

### Certification Start
- [ ] Begin department certification path
- [ ] Schedule quiz for Module E1/O1/etc.

---

## Week 2 — Certification

### Complete Required Modules
- [ ] Score ≥ 80% on Architecture Quiz (E1)
- [ ] Complete department-specific module
- [ ] INBOX routing quiz ≥ 90%

### First Solo Project
- [ ] Assigned by department lead
- [ ] Tracked in Honey-Do
- [ ] Completed within estimate

---

## 30-Day Milestones

| Day | Checkpoint |
|---|---|
| 7 | INBOX routing correct, no P1 mistakes |
| 14 | First certification module complete |
| 21 | Cross-department awareness session done |
| 30 | Full department certification in progress |
