# BX3 Paper Review — v1

## Scope
This directory holds extended, standardized v1 review PDFs.

## Standard Structure (per paper)
1. Title + subtitle
2. Abstract (250 words)
3. Authors + affiliations + ORCID
4. Keywords (6-8)
5. 1. Introduction (1.5-2 pages)
6. 2. Background & Related Work (3-4 pages)
7. 3. Architecture / Framework (4-6 pages)
8. 4. Formal Specification (2-3 pages)
9. 5. Implementation & Validation (2-3 pages)
10. 6. Discussion (1-2 pages)
11. 7. Conclusion
12. 8. Acknowledgments
13. References
14. Peer Review Instructions (appendix)

## Visual Requirements per Paper
- Figure 1: Conceptual architecture diagram
- Figure 2: System flow / sequence diagram
- Figure 3: Data flow or state diagram
- Figure 4: Application domain example
