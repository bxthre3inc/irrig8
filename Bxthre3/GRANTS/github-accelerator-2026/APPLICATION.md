## Project Name
**Investor Protector**

## Tagline
Open-source transparency layer for AI-native companies — real-time velocity, verified metrics, auditable truth.

## Problem
AI startups are lying to investors. 

Not intentionally (mostly). But when a founder says "we're shipping fast," what does that mean? Commits? Lines of code? Features? When they say "burn rate," is that AWS, payroll, or the founder's lifestyle? When they say "traction," is it real users or bot traffic?

Investors have no standardized way to verify claims. Due diligence takes weeks. Founders game the metrics. The result: bad investments, wasted capital, and a market where appearance beats reality.

## Solution
Investor Protector is an open-source dashboard that pulls real, verifiable data from a company's actual systems:

- **GitHub API** → real commit velocity, contributor activity, repository health
- **Zo Computer usage logs** → actual infrastructure costs (we run on $18/mo, not $5K/mo)
- **File system scans** → deliverables count, documentation quality, code-to-doc ratio
- **Verification engine** → every claim tied to a hash, every metric auditable

Investors get a URL: `company.io/investor` → live metrics, no founder interpretation.

## Why Open Source?
Because trust requires transparency. If the verification layer itself is proprietary, we're back to square one. By open-sourcing Investor Protector, we:
- Let investors self-host (trust no third party)
- Let founders customize (their data, their dashboard)
- Let the community audit (the code is the guarantee)
