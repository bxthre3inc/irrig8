# 2B2F Council — Internal Coordination Protocol

## Purpose
Self-organizing leadership structure for 120-month plan development.

## Agent Structure
```
ARCHIVIST (Bxthre3 Strategic Planning Unit)
├── Reports to: Council (collective)
├── Escalates to: brodiblanco (monthly overwatch)
├── Synthesizes: All Council outputs
├── Maintains: MASTER_TIMELINE, methodology, final paper formatting
└── Co-authors: Research paper per Council mandate
```

## Communication Flows
```
ARCHIVIST ←→ COUNCIL (4x daily)
    ↓
Monthly Synthesis
    ↓
brodiblanco (1x monthly review)
    ↓
[Escalation loop if required]
```

## Meetings

### Daily Council Sync (4x)
**Purpose:** Status, blockers, task distribution  
**Format:** Async log entries to SYNC_LOGS/  
**Responsible:** ARCHIVIST initiates, Council responds

### Monthly Lead Author Review
**Purpose:** Direction validation, milestone check, resource alignment  
**Format:** Email summary + any sync requests  
**Responsible:** ARCHIVIST compiles, brodiblanco reviews

## Decision Authority Levels

| Level | Authority | Examples |
|-------|-----------|----------|
| L1 | Council consensus | methodology choices, scenario selection |
| L2 | ARCHIVIST operational | file naming, timeline formatting, daily prioritization |
| L3 | brodiblanco override | direction changes, timeline extensions, publication venue |

## Zero Budget Constraints
- No paid tools (open source only)
- No external contractors (Council-only labor)
- No marketing spend (public research model)
- No travel (async collaboration)

1. **M12:** Published co-authored paper ✓
2. **M12:** Complete 120-month execution plan ✓
3. **M24:** 1% GDP deployment path validated ✓
4. **M36:** 5% GDP scaling model confirmed ✓
5. **M120:** 60%+ GDP trajectory documented ✓
