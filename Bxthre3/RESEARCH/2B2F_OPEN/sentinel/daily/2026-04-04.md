## Research Notes

### Key Insight: The "Utility Token" Viability Window

Recent SEC enforcement patterns suggest that pure utility token designs — those with no profit expectation, no common enterprise, and genuine consumptive use — remain viable but require rigorous documentation.

**Critical design elements identified:**
- Token must be consumptive (burned/used, not held for appreciation)
- No marketing of returns or yield
- Decentralized governance with no controlling entity
- Open-source code with transparent functionality

### Jurisdiction Watch: Switzerland

Swiss FINMA guidelines provide the clearest token classification framework globally. Worth deeper analysis for potential entity establishment.
