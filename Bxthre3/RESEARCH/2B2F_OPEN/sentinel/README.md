## Research Principles

1. **Zero Budget:** Public sources only — academic papers, regulatory filings, open data
2. **Simulation-First:** Model before deployment
3. **Risk-Aware:** Document every failure mode before it occurs
4. **Council-Based:** Distributed validation, consensus-driven
