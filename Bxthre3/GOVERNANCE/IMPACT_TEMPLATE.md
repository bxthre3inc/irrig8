# Bxthre3 Net Employment Impact Template

## FORMAT REQUIREMENT: Each impact assessment must include a concrete EXAMPLE

---

## 1. Jobs Created (Direct Hires)

**Definition:** New W-2 or 1099 positions directly funded by this project

**EXAMPLE for Autodark:**
```
- 7 Assembly Technicians @ $48,000 = $336,000
- 3 Quality Control Inspectors @ $52,000 = $156,000  
- 2 Composite Fabricators @ $62,000 = $124,000
- 1 Production Manager @ $78,000 = $78,000

Total Created: 13 positions
```

---

## 2. Jobs Displaced (Automation/Replacement)

**Definition:** Roles eliminated or reduced due to technology, efficiency, or offshoring

**EXAMPLE for Autodark:**
```
- 4 Manual assembly roles → automated pick/place systems
- 1 Inventory clerk → RFID tracking automation
- 2 QC inspectors → computer vision systems (net: -1 after retraining 1)

Total Displaced: 4-6 positions (depending on retraining success)
```

---

## 3. Jobs Protected (Second-Order Effect)

**Definition:** Existing roles that would be lost without this project

**EXAMPLE for Irrig8:**
```
- 120 farming operations that would fail without precision irrigation
- Estimated 450 farm worker jobs protected = ~$13.5M in payroll preserved

Protected count: 450 (not net new, but preserved)
```

---

## 4. Net Employment Calculation

```
Jobs Created:     +13
Jobs Displaced:   -4  
Retraining:       +1 (1 displaced worker moved to new role)
Net Direct:     +10

Jobs Protected:   +450 (reported separately as "preserved")
```

---

## 5. Impact Verification Methods

| Method | Tool/Source | Frequency |
|--------|-------------|-----------|
| Payroll | Gusto API | Monthly |
| Displacement | Internal HR tracked | Quarterly |
| Protection | Customer survey + USDA data | Annual |
| Third-party | B-Corp audit | Bi-annual |

---

## REQUIRED FIELDS

Every project impact doc MUST include:
- [ ] Example for each category
- [ ] Concrete numbers with timestamps
- [ ] Net calculation showing +/-
- [ ] Verification method specified
