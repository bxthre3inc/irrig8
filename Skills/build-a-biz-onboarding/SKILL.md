# Build-A-Biz Onboarding Portal Skill

## Overview
This skill powers the Valley Build-A-Biz LLC client onboarding portal at `/onboarding/build-a-biz`.

## Features
- Password-protected page (password: 1991)
- 7-section form wizard with progress bar
- Auto-save functionality
- Session management (1-hour expiry)
- JSON data export

## Data Storage
Client data saved to: `Bxthre3/projects/build-a-biz-llc/client-data/{client-name}-onboarding.json`

## Sections
1. Personal Info
2. Business Info
3. Services
4. Target Market
5. Business Story
6. Financial Info
7. Legal & Submission
