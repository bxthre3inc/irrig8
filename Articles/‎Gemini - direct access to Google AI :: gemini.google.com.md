-----------------------------------------------

Every entity in the ecosystem (a "Node") must operate using the same three logical interfaces. This ensures that a "Sensor Hub" and a "Regional Department" can be addressed using the same code logic.

* **Egress (The Actuators):** The output. This is the execution of a command (e.g., opening a valve, or sending a wire transfer).

3. The "Worksheet" Protocol
---------------------------

The secret to modularity is the **Over-The-Air (OTA) Worksheet**.

Instead of programming a node to "do X," you provide it with a math-based worksheet.

* **Decoupling:** The Node doesn't need to know *why* it's doing something; it only needs to satisfy the equation in its current worksheet.
* **Efficiency:** High-compute number crunching (like soil variability mapping or market trend analysis) happens at L2 or L3. Only the *result* (the logic parameters) is sent back down to the Hub.
* **Auditability:** Every action taken by a node is a direct result of its current worksheet and its sensor input, making the entire business 100% auditable and deterministic.
